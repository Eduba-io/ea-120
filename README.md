# Flashpoint × Eduba

A single-page landing site prepared for the Flashpoint product and intel research team, designed to be opened on a phone at Booth 360 during the eMerge Americas 2026 conference pitch. It carries the Eduba frame (60 / 30 / 10), the VigilOre multi-agent compliance case study translated into analyst-investigation vocabulary, the ICM paper, Jake's credibility, and a direct calendar CTA, rendered in Flashpoint's light-mode orange-and-charcoal visual register (Inter headings, IBM Plex Mono for labels) so it reads as native to the flashpoint.io web property.
